# Feishu Fields

This prototype only uses the smallest practical Feishu fields.

## Recommended columns

- Candidate Name
- Role
- Current Stage
- Owner
- Next Step
- Status
- Last Update
- Notes

## Fixed stage values

- 待补资料
- 待约初面
- 待面试反馈
- 待继续推进
- 待归档
- 已暂停
- 已转岗

## Fixed status values

- 正常推进
- 临近超时
- 已卡住

## Time rule

- within 24 hours → 正常推进
- 24–48 hours → 临近超时
- over 48 hours → 已卡住

## Candidate reply summary format

Use this 3-line format by default:

- 已确认：
- 候选人意向/限制：
- 下一步：

## Example write-back

- Candidate Name: 陈玉玲
- Role: 资深后端研发工程师（电商 / 搜索方向）
- Current Stage: 待约初面
- Owner: HR
- Next Step: 发出初面邀请
- Status: 正常推进
- Last Update: 2026-03-15
- Notes:
  - 已确认：候选人愿意继续推进，可进入初面。
  - 候选人意向/限制：时间以工作日白天为主。
  - 下一步：协调面试官时间并发出初面邀请。
