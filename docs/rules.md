# Fixed Rules

## Stage fields

- 待补资料
- 待约初面
- 待面试反馈
- 待继续推进
- 待归档
- 已暂停
- 已转岗

## Status fields

- 正常推进
- 临近超时
- 已卡住

## Time rules

- within 24 hours → 正常推进
- 24–48 hours → 临近超时
- over 48 hours → 已卡住

## Default communication style

- concise
- human
- judgment first, then reason, then next step
- no long reports
- no table voice
